Jeu de Données : UR Fall Detection Dataset

Ce jeu de données est conçu pour la recherche sur la détection de chutes à l'aide de vidéos. Il est utilisé dans de nombreux projets de vision par ordinateur et d'intelligence artificielle pour reconnaître des chutes chez les personnes âgées ou vulnérables.

Contenu du Dataset

  • urfall_camo_adls : contient des vidéos d'activités quotidiennes (ADL = Activities of Daily Living), telles que marcher, s'asseoir, se lever, ramasser un objet, etc.
  • urfall_camo_falls : contient des vidéos simulant des chutes dans divers scénarios.

Accès au Dataset

Vous pouvez téléchargé directement depuis le site officiel de l'Université Rzeszów le jeu de donnée :

https://fenix.ur.edu.pl/~mkepski/ds/uf.html

Utilisation

Les vidéos sont accompagnées de fichiers de métadonnées et sont utiles pour nous permetre :

  • Analyser les mouvements humains
  • Détecter des anomalies comme les chutes
  • Entraîner des modèles de classification de posture

Donc pour ce projet academique nous utiliserons ce jeu de donnée
